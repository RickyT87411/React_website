### JSX の中にコメントを記述するには？ {#how-can-i-write-comments-in-jsx}

```jsx
<div>
  {/* Comment goes here */}
  Hello, {name}!
</div>
```

```jsx
<div>
  {/* It also works 
  for multi-line comments. */}
  Hello, {name}! 
</div>
```
